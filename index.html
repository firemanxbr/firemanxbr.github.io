Flask Web Development a great book

I was reading the excellent book “Flask Web Development” written by Miguel Grinberg and realized that on page 4 the book has no information of how to install the package python-virtualenv for Fedora 20/Red Hat Enterprise Linux/CentOs. So I leave here my contribution to all of you:

$ sudo yum install python-virtualenv

Currently on Fedora 20 he’s in your version:

$ virtualenv--version
1.10.1

Creating git branches in GitHub

When I migrated this blog from Tumblr to GitHub Pages I had to sort out some issues. While searching about how to have them sorted out I thought: why not share this knowledge with other folks?

TIP: I created a new version of my application, but I have not had it versioned before:

Initialize versioning in my local source:

$ git init

Create my new branch:

$ git checkout -b name-new-branch

Add my sources in this initial version:

$ git add .

Commit of initial version:

$ git commit -m "Initial version of branch 'name-new-branch' "

And send to GitHub(create new branch and push the initial version):

$ git push URL-YOUR-PROJECT-GITHUB.git name-new-branch
