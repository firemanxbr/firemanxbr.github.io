When I migrated this blog from Tumblr to GitHub Pages I had to sort out some issues. While searching about how to have them sorted out I thought: why not share this knowledge with other folks?

TIP: I created a new version of my application, but I have not had it versioned before:

I have initialize versioning in my source:

$ git init

Create my new branch, only local in this moment:

$ git checkout -b name-new-branch

Add my sources in this initial version:

$ git add .

After I have commit this initial version:

$ git commit -m "Initial version of branch 'name-new-branch' "

And sent to GitHub(create new branch and push this initial version):

$ git push URL-YOUR-PROJECT-GITHUB.git name-new-branch
